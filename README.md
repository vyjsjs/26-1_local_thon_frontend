# 26-1_local_thon_frontend

수원 공방거리 NFC 스탬프 투어 웹 앱 프론트엔드입니다.

## 기술 스택

- Next.js 16 (App Router)
- React 19
- TypeScript
- Tailwind CSS 4
- shadcn/ui

## 실행 방법 (npm)

```bash
cd 26-1_local_thon_frontend
npm install
npm run dev
```

개발 서버: [http://localhost:3000](http://localhost:3000)

## 빌드

```bash
npm run build
npm start
```

## 디렉터리 구조

```
app/          # 페이지 라우트
components/   # UI·도메인 컴포넌트
hooks/        # 커스텀 훅
lib/          # 데이터·유틸·i18n
public/       # 정적 에셋
```

디자인 토큰은 `DESIGN.md`를 참고하세요.
